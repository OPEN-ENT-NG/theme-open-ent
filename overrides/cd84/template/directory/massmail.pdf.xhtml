La première connexion à NEO comprend une étape préalable indispensable, qui vous permet de personnaliser vos informations de connexion.

  • De quoi se munir ?

    Pour se connecter à NEO, vous devez vous munir de votre adresse E-mail personnelle et de cette feuille.

  • Comment accéder à NEO ?

    Vous pouvez accéder à NEO en passant par le site vitrine : aucollege84.vaucluse.fr.

  • Comment se connecter la première fois ?

    Utiliser les identifiants ci-dessous sur la page de connexion :
    • Identifiant : {{login}}
    • Code d'activation {{activationCode}}

Pour sécuriser vos accès, il vous sera demandé de choisir un nouveau mot de passe, d’indiquer votre adresse E-mail personnelle et si vous le souhaitez de saisir un numéro de téléphone portable pour récupération du mot de passe en cas d’oubli. Vous aurez aussi la possibilité de consulter la charte de NEO et l’obligation d’en accepter les termes.

Nous vous recommandons de mélanger chiffres, lettres, majuscules, minuscules et caractères spéciaux lors du choix de votre mot de passe personnel.

Désormais, vous pouvez vous connecter à NEO en utilisant votre nouveau mot de passe.
