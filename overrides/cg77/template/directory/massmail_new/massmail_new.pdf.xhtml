» Comment me connecter ?

Vous possédez déjà un compte sur l’espace numérique. Il vous suffit de saisir votre identifiant habituel et le mot de passe défini lors de votre première connexion.
Pour rappel, votre identifiant est le suivant :

{{login}}


» 3 étapes pour réinitialiser mon mot de passe

  1. En cas d'oubli de votre mot de passe, rendez-vous sur https://ent.seine-et-marne.fr et cliquez sur « Se connecter à l'ENT », ou téléchargez l'application mobile ENT 77:
  2. Cliquez sur « identifiant ou mot de passe oublié ». Un mail sera envoyé à l'adresse indiquée lors de l'activation de votre compte.
  3. Ouvrez le mail et cliquez ensuite sur le lien pour définir votre nouveau mot de passe.

» Mode d'emploi de ma première connexion

  1. Pour vous connecter, rendez-vous sur https://ent.seine-et-marne.fr, ou téléchargez l'application mobile ENT 77:
  2. Entrez votre identifiant (à conserver pour vous connecter ultérieurement) et votre code d'activation :

    Identifiant : {{login}}

    Code d'activation : {{activationCode}}

  3. Votre mot de passe doit être composé d'un minima de 12 caractères combinant majuscules, minuscules, chiffres et caractères spéciaux.
  4. Renseignez votre adresse mail personnelle pour obtenir un mail de réinitialisation en cas d'oubli de votre mot de passe.
  5. Vous n'avez plus qu'à lire et accepter les Conditions Générales d'Utilisation ainsi que la charte d'utilisation.
